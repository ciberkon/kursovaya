# Разработка алгоритма компьютерного зрения для обнаружения подводных трубопроводов в симуляторе АНПА 

Программа для автономного необитаемого подводного аппарата (АНПА). Система реализует интеллектуальный поиск и захват подводных трубопроводов в реальном времени с использованием нейросетевого детектора **YOLOv8**[cite: 3].

## Возможности
*   **Вычисление координат:** Автоматический расчет центра объекта[cite: 3].
*   **Визуализация:** Отрисовка рамок (bounding boxes), вывод текущего показателя FPS для контроля производительности, а также количества объектво на экране[cite: 3].
*   **Дообучение** При необходимости есть возможность добавить данных и метрик к ним и продолжить обучение модели[cite: 3].

## Быстрый старт
1.  Установите необходимые Python-зависимости:
    ```bash
    python -m pip install ultralytics opencv-python numpy
    ```
2. Запустите алгоритм из директории:
    ```bash
    python detection.py
    ```

## Настройка параметров
В основном программном модуле вы можете отрегулировать следующие параметры:

| Параметр | Значение по умолчанию | Описание |
| :--- | :--- | :--- |
| `conf` | 0.5 | Порог уверенности нейросети (чем ниже, тем чувствительнее захват)[cite: 4] |
| `model` | best.pt | Версия предобученных весов[cite: 3] |

А также указать пути к другой обученной модели или к другом видеопотоку.

## Состав проекта
*   **detection.py** — основной программный модуль детекции.
*   **train.py** - модуль обучения нейросети
*   **data.yaml** - указания для обучения
*   **Пояснительная записка** — полная документация курсового проекта ( листов).
*   **Приложение А** — руководство пользователя.
*   **Приложение Б** — листинг кода с комментариями.
